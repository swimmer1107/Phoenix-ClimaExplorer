# ⚛️ PyClimaExplorer Frontend

The frontend for PyClimaExplorer is a modern, high-performance web application built with **React 19** and **Vite**, designed for immersive climate data visualization.

---

## 🎨 Key UI Components

*   **Globe Visualization**: A 3D interactive globe powered by `Three.js` and `React Three Fiber`.
*   **Climate Dashboard**: Interactive charts using `Recharts`.
*   **Aesthetics**: Glassmorphism and dark mode styling with `Tailwind CSS`.
*   **Animations**: Liquid transitions and micro-interactions using `Framer Motion`.

## 🛠 Features

- **Real-time API Integration**: Connects seamlessly with the FastAPI backend.
- **Dynamic Routing**: Smooth navigation between Landing, Live Globe, and Analytics pages.
- **Responsive Design**: Optimized for multiple screen sizes.

---

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

---

## 📂 Structure

- `src/components`: Reusable UI elements (Buttons, Cards, Modals).
- `src/pages`: Main view components (GlobeView, LandingPage, Dashboard).
- `src/services`: API client and data fetching logic.
- `src/hooks`: Custom React hooks for state and lifecycle management.
